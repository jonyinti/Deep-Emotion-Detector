# 😲 Streamlit based Deep Emotion Detector 😄

[![Project Status: Active](https://www.repostatus.org/badges/latest/active.svg)](https://www.repostatus.org/#active)
[![Maintained by Jony](https://img.shields.io/badge/Maintained%20by-Jony-blue.svg)](https://github.com/jonyinti)

A Streamlit-based web application to process text and detect emotions using the **EmoRoBERTa** model from Hugging Face Transformers 🤗.

## 🚀 Installation

```bash
pip install -r requirements.txt
```

## 📖 Usage

Clone the repository:

```bash
git clone https://github.com/jonyinti/Deep-Emotion-Detector.git
```

Go to the project folder:

```bash
cd Deep-Emotion-Detector
```

Run the application:

```bash
streamlit run app.py
```

Open your browser:

```
http://localhost:8501
```

## 🛠️ Technologies Used

- Python
- Streamlit
- Hugging Face Transformers
- PyTorch
